STRUCTURE:

<h1> : page title
<p class="subhead"> : subheading
<div class="header-authors"> : authors grid
<div class="header-published"> : published date information
<div class="contents box"> : table of contents
<section id="introduction"> : sections from table of contents
<p class="cite-parent"> : allows to add citations
<figure class="text-center"> : for images with caption.
<p class="inline-math"> : allows to handle mathjax expressions
    Use $$ – some equation here – $$ to add an expression
    <p class="math-left"> and <p class="math-right"> : makes left/right alignment of an expression
<div class="static"> : supports static illustrations with caption
<div class="framed"> : frame to insert interactive elements and scripts
<div class="formula-wrapper"> : allows adding content into styled and responsive table
<section class="references"> : created section of references.
